Abstract

Federated Learning (FL) enables collaborative training without exposing raw data, yet remains vulnerable to gradient inversion attacks that recover private inputs from shared gradients. We present GI-DQA, the first attack that successfully reconstructs sensitive document question-answering inputs by combining optimization-based image recovery with analytic extraction of question–answer pairs.

Key Contributions

Novel Attack

First gradient inversion attack targeting multimodal Document QA models.

Hybrid Strategy

Visual pixel optimization + analytic question–answer reconstruction.

Comprehensive Evaluation

Tested on Donut-Base and LayoutLMv3-Base using the privacy-aware PFL-DocVQA dataset.

Method

GI-DQA takes leaked gradients (∇θ), public document templates, and model weights as input. It extracts the exact question & answer tokens analytically, then optimizes document pixels to match client gradients while respecting visual priors.

Results

GI-DQA dramatically outperforms prior inversion methods on multimodal models.

Recovered private fields (names, IDs) using GI-DQA vs. prior attacks.
